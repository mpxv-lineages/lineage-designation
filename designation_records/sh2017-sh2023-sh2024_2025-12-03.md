# Adding definitions for sustained outbreaks `IIb/sh2017`, `Ib/sh2023`, and `Ia/sh2024`

In line with the paper _A systematic nomenclature for mpox viruses causing outbreaks with sustained human-to-human transmission_ (<https://www.nature.com/articles/s41591-025-03820-6>), we are adding definitions of the 3 sustained outbreaks `IIb/sh2017`, `Ib/sh2023`, and `Ia/sh2024` to this repository.

Currently, only `IIb/sh2017` has sublineages defined - but the structure is in place to add sublineages for `Ib/sh2023` and `Ia/sh2024` in the future as needed.

All definitions have been been moved to the `definitions` folder, which contains subfolders for each clade. Each sustained outbreak definition is in its own yaml file within the appropriate clade folder. This results in the following structure:

```txt
definitions/
    IIb/
        sh2017.yml
        sh2017/
            A.yml
            A.1.yml
            ...
    Ib/
        sh2023.yml
    Ia/
        sh2024.yml
```

For now, the auto-generated files will remain scoped to `IIb/sh2017`.
